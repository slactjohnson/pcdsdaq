Simulated DAQ¶

If pydaq is unavailable or the real DAQ nodes are occupied, you can still test your code against a simulated daq module.

Simply do:

from pcdsdaq.sim import set_sim_mode
set_sim_mode(True)
pcdsdaq.sim.set_sim_mode(sim_mode)¶
Parameters:sim_mode (bool) – If True, we’ll set the Daq class to run in simulated mode. If False, we’ll attempt to import pydaq to run the Daq in real mode.
